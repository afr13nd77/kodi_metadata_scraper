# BUG-008: getepisodelist падает при отсутствии kinopoisk_id в episodeguide (foreign scraper migration)

**Дата обнаружения:** 17.07.2026
**Версия:** 3.17.0 / 3.17.0
**Серьёзность:** major
**Статус:** done (v3.17.2)

---

## Наблюдаемое поведение

При обновлении информации о сериале Silo (kp_id=4541515) на боевой установке Kodi вызывает `getepisodelist` с episodeguide URL вида:
```json
{"tmdb": "125988", "imdb": "tt14688458", "tvdb": "403245"}
```
Скрапер не находит `kinopoisk_id` в guide и возвращает ошибку:
```
ERROR: _handle_getepisodelist: no kinopoisk_id in guide
WARNING: No information found for item 'D:\TV Shows\Silo\Season 03\'
```

## Ожидаемое поведение

Скрапер должен резолвить KP ID из IMDB ID (который присутствует в guide) через существующий механизм `search_kp_by_imdb()` (KP API → Wikidata fallback) и загрузить список сезонов/эпизодов.

## Шаги воспроизведения

1. Добавить сериал в библиотеку Kodi через другой скрапер (TMDb, TVDB).
2. Переключить scraper на metadata.tvshows.ums.
3. Запустить «Обновить информацию» / автосканирование.
4. Kodi вызывает `getepisodelist` напрямую (без `find` → `getdetails`), передавая исходный episodeguide с TMDB/TVDB/IMDB ID, но без `kinopoisk_id`.
5. Скрапер не может загрузить сезоны.

## Окружение

- **Боевая установка (kodi-1.log):** Kodi 21.2, Windows 10, metadata.tvshows.ums v3.17.0
- **Тестовая установка (kodi-2.log):** Kodi 21.3, Windows 10, metadata.tvshows.ums v3.15.3 — работает корректно (полный цикл find→getdetails)

## Root cause

`_handle_getepisodelist` (tv_scraper.py:713-716) проверяет `guide.get("kinopoisk_id", 0)` и при нулевом значении делает `return`. Guide от чужого скрапера содержит ключи `tmdb`, `imdb`, `tvdb` — без `kinopoisk_id`. Механизм резолва IMDB→KP (`search_kp_by_imdb()` из `shared/utils.py`) уже реализован (BUG-007), но не подключён к `_handle_getepisodelist`.

## Решение

В `_handle_getepisodelist`, когда `kinopoisk_id` отсутствует но есть `imdb` (ключ чужих скраперов) или `imdb_id` (наш формат):
1. Извлечь IMDB ID из guide (`guide.get("imdb")` или `guide.get("imdb_id")`).
2. Вызвать `search_kp_by_imdb(imdb_id, settings, logger)` для резолва KP ID.
3. При успехе — продолжить загрузку сезонов с полученным `kp_id`.
4. При неудаче — логировать ошибку и вернуть пустой результат.
