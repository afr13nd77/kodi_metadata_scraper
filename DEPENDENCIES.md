# Зависимости проекта

Краткий ответ: для использования аддона достаточно установить **Kodi**. Никаких дополнительных пакетов устанавливать не нужно.

---

## Среда выполнения

| Компонент | Версия | Способ получения |
|-----------|--------|------------------|
| Python | 3.8 | Встроен в Kodi v20 Nexus / v21 Omega — **не устанавливается отдельно** |

Аддон не использует никаких внешних pip-пакетов. Всё необходимое уже есть в стандартной библиотеке Python и в модулях Kodi.

---

## Зависимости аддона (addon.xml)

Объявлены в `addon.xml` каждого аддона. Оба компонента встроены в Kodi:

| Зависимость | Минимальная версия | Примечание |
|-------------|-------------------|------------|
| `xbmc.metadata` | 2.1.0 | Интерфейс скрапера метаданных |
| `xbmc.python` | 3.0.0 | Python-биндинги Kodi |

---

## Модули стандартной библиотеки Python

Используются только встроенные модули Python 3.8 — никакой установки не требуется:

| Модуль | Назначение |
|--------|-----------|
| `json` | Разбор и сериализация JSON |
| `urllib.request`, `urllib.parse`, `urllib.error` | HTTP-запросы к внешним API |
| `re` | Регулярные выражения |

| `threading` | Потокобезопасное кэширование (Lock) |
| `time` | Временные метки, ограничение частоты запросов |
| `hashlib` | Генерация ключей кэша (md5) |
| `dataclasses` | Модели данных |
| `difflib` | Нечёткое сравнение строк (SequenceMatcher) |
| `html` | Декодирование HTML-сущностей |
| `traceback` | Логирование ошибок |
| `math` | Вычисления |
| `typing` | Аннотации типов |

## Модули Kodi Python API

Предоставляются самим Kodi — доступны внутри любого аддона:

| Модуль | Назначение |
|--------|-----------|
| `xbmc` | Логирование, настройки, информация о плеере |
| `xbmcaddon` | Метаданные аддона, доступ к настройкам |
| `xbmcgui` | Диалоги и элементы UI (ListItem, Dialog) |
| `xbmcplugin` | Точки входа плагина, элементы списков |
| `xbmcvfs` | Виртуальная файловая система (хранение кэша) |

## Внешние API (сетевые, настраиваются пользователем)

Аддон обращается к внешним сервисам. Все URL конфигурируются через настройки аддона в Kodi.

### Kinopoisk Unofficial API

- **Адрес:** `kinopoiskapiunofficial.tech`
- **Назначение:** Основной источник метаданных (названия, описания, постеры, рейтинги)
- **API-ключ:** Обязателен. Регистрация на сайте сервиса, бесплатный тариф доступен.
- **Ограничение:** 20 запросов в секунду на бесплатном тарифе.
- **Обязателен:** Да — без ключа аддон не работает.

### OMDb API

- **Адрес:** `omdbapi.com`
- **Назначение:** Агрегированные рейтинги (IMDb, Rotten Tomatoes, Metacritic)
- **API-ключ:** Обязателен. Бесплатный тариф доступен на сайте сервиса.
- **Примечание:** В ряде регионов может потребоваться VPN.
- **Обязателен:** Нет — опционально, включается в настройках аддона.

### TVMaze API

- **Адрес:** `api.tvmaze.com`
- **Назначение:** Описания эпизодов на английском языке
- **API-ключ:** Не требуется. Сервис бесплатный.
- **Примечание:** В ряде регионов может потребоваться VPN.
- **Обязателен:** Нет — опционально, включается в настройках аддона.

## Зависимости для разработки

Нужны только при разработке и запуске тестов. Пользователям Kodi не требуются.

| Компонент | Назначение |
|-----------|-----------|
| `pytest` | Запуск тестов (534 теста) |
| Python 3.8+ venv | Изолированное окружение для разработки |

Установка окружения для разработки:

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install pytest
pytest
```
